Search Arabidopsis thaliana DNA Sequences

This program doesn't work with some versions of Netscape. Use Internet Explorer or Netscape 6.2.

Updated August 1, 2002.

This program was created by Todd Richmond.

For the most up-to-date annotation of a given EST sequence from the AFGC arrays, please check the CLONE SEARCH page at TAIR or BLAST the sequence against GenBank or the databses at TAIR or TIGR.

Note: TAIR is developing a parallel resource to the AFGC Clone List. Please check the TAIR news page for information about this development (August 1, 2002).

This page offers access to information for 123,000+ Arabidopsis DNA sequences (all Arabidopsis sequences less than 20,000 bp that are not genome survey sequences). Enter search criteria into one or more of the following categories and press 'Submit Query' to search. Entering terms into more than one box will search for sequences that match all of the criteria. The information returned will include GenBank accession number, clone id, GID, definition lines, putative function as assigned by TIGR, and TIGR's BAC and chromosome locus (as determined by DNA sequence similarity). Match length, percent identity and availibility of microarray data from the Stanford Microarray Database are also given. For those ESTs represented on the AFGC microarrays, the icon will take you to the SPOT HISTORY page and the icon will take you to the EXPRESSION VIEWER page. From the results page, you can find all sequences that belong to the same BAC locus/chromosomal locus or genome region.

CloneID (i.e. H10C11T7, 600040331R1)
Accession (i.e. AA042439, Z18792, T04072)
GID (i.e. 8720447, g2446148)
TIGR BAC Locus (i.e. F2K13_60)
TIGR Chromosome Locus (i.e. At5g16910)
Function (i.e. cellulose synthase, kinase, syntaxin)
GenBank Definition Line (i.e. terms from "BG459617 M78N08STM Arabidopsis developing seed Arabidopsis thaliana cDNA clone 600040331R1 5', mRNA sequence")

Search hints

  • Search for multiple items within each category by separating search terms with a space.
  • Searches for GenBank Definition Line and Function support partial word matches. A search for "pyruvate" will match "phosphoenolypyruvate carboxylase", "beta-alanine-pyruvate aminotransferase", "hydroxypyruvate reductase" (and about 600 other ESTs). If you get too many hits, try being more specific.
  • Searches are case-insensitive.
  • Searches for CloneID, Accession, GID, BAC Locus or Chromosome Locus will be faster than searches for GenBank Definition Line or Function.
  • Defining what constitutes a CloneID can be tricky. If a search on CloneID fails, try searching the GenBank Definition Line with the same search term.
  • Searches for common terms (i.e. kinase) or for genes that may have a lot of ESTs (i.e. rubisco) may produce large result sets. Be patient as it may take a minute or two to find and load all of the data.
  • Use common sense when specifying multiple criteria. Specifying an Accession for one EST and a GID for another EST will return no information for either EST.

Details of Sequence Similarity Searches

Similarity searches were done using SSAHA2 software (Sequence Search and Alignment by Hashing Algorithm; available from the Sanger Centre). This software does very rapid searches for nearly identical matches. A comparison of 113,000 sequences against the entire Arabidopsis genome takes approximately 40 minutes on a Dell PowerEdge server running RedHat Linux 2.2. The following parameters were used:

-maxGap 5 Maximum gap allowed between successive hits for them to count as part of the same match.
-maxInsert 5 Maximum number of insertions/deletions allowed between successive hits for them to count as part of the same match.
-wordLength 10 Size in base pairs of the words used to form the hash table.
-minPrint 50 The minimum number of matching bases or residues that must be found in the query and subject sequences before they are considered as a match and thus printed.
-sortMatches 1 Output only the top n matches for each query, sorted by number of matching bases, then by subject name, then by start position in the query sequence.
-parserFriendly (a.k.a. perlFriendly) Show one match per line as a set of tab delimited fields

The output of SSAHA2 was compared to the list of sequence features extracted from TIGR's XML annotation files of the Arabidopsis genome and used to assign a TIGR feature to each sequence (if possible). The minimum criteria is a match of 50 basepairs with 90+% identity. For the most part, most matches are 100% due to the parameters set above. Be cautious, however, when using short matches to decide that an EST belongs to any one particular gene. When a match to a TIGR feature was not found, the interval into which it fell was recorded. If your sequence of interest falls into one of these intervals, you can initiate a search to find other sequences that fall into that same interval. Since some of these regions are quite large, there is no guarantee that all sequence in the same unannotated region belong to the same coding sequence.

Four passes were made through the sequences to determine sequence similarity. First, 123,527 Arabidopsis sequences were compared to the entire genome. Arabidopsis sequences less than 50 basepairs in length and GSS sequences were ignored. Those without good matches were then compared to the Arabidopsis chloroplast and mitochondrial genomes (697 sequences). Then against vector sequences from GenBank (38 sequences) and finally human mRNA (18 sequences). That left 5554 sequences with no sequence similarity information. Using a slower, but more forgiving BLASTN search, several thousand of these sequences were assigned positions. A total of 2379 of the original 123,527 sequences (1.9%) have no match to the completed genome using the criteria above. Of the remaining 98%, 19,582 sequences (16%) have questionable matches - they have match lengths less than 100 basepairs, or sequence identity less than 95%.
