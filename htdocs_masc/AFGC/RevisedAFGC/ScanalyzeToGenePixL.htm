The Switch from Scanalyze to Genepix

During the processing of Cycle 1 AFGC experiments, new analysis software (Genepix Pro 3.0) was implemented, for the purpose of getting more accurate results and speeding up the processing time of the slides. Before this software was implemented all of the experiments were processed using a program called Scanalyze2. All experiments done after Cycle 1 will be analyzed using the Genepix Pro 3.0 software. The following is a comparison between the Scanalyze2 software and the Genepix Pro 3.0 software, and a list of experiments that were reanalyzed and reposted with the new data to the SMD.

Genepix vs. Scanalyze

Genepix Pro 3.0Scanalyze 2
  1. Automatic spot finding feature allows for faster gridding
  2. Easier to see interfering background, making flagging more accurate (see flagging guide)
  3. Tighter fit of grids, allows for more accurate data collection
  4. Can see and flag saturated spots
  5. Ability to do better quality control checks of data before loading to SMD
  1. Grids must be adjusted on a spot by spot basis to get the most accurate fit,
  2. Interfering background hard to see, and thus some spots were not flagged properly
  3. Grids did not always fit tightly around spots, so the data is not as accurate
  4. Saturated spots are not apparent
  5. Quality control had to be done on experiments after the experiment was already loaded in SMD
